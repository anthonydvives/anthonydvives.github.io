Discord Bot

This is a bot written using the Discord Java API to write a Discord Bot for the popular text and voice chat. This was developed during my Sophmore year of college and has been slowly updated throughout the years. I host the bot on my personal desktop and frequently use the bot in my own Discord chat server.

Quotes

The bot has the ability to store quotes given by using the trigger word "d!" followed by by the command "quote" + a number. The same can be done to add quotes or delete quotes by using "d!a-quote" + the quote or "d!d-quote" followed by the corresponding number associated with quote to be deleted.

Message Purging & Magic 8 Ball

The bot also has the ability to delete the message history of the text channel the command was called in. The bot can also function as a magic 8 ball and answer yes or no questions.

Petri Net

This is a project written for my SE-306 Formal Methods course to develop a working petri net generator with a working GUI that allows the user to input the desired places and transitions and see the reachability tree for the given input.

Input Screen

This is the view seen after inputting the desired number of places & transitions. A table is generated and The user is required to input the number of tokens during each Input for each transtion and each Output for each transition.

Output Message

Finally, the inital markings are required to be inputted and the output for the data set can be generated and written to file.
